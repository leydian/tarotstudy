# 🎴 Tarot Codex (타로 코덱스)

**Tarot Codex**는 타로 카드의 깊은 상징과 철학적 의미를 탐구하고자 하는 초보자부터 전문가까지 모두를 위한 초고밀도 타로 라이브러리 및 리딩 엔진입니다.

## ✨ 주요 특징

### 1. 78장 초심화 카드 라이브러리
- **전 카드 구축 완료**: 메이저 아르카나 22장 및 마이너 아르카나 56장(Wands, Cups, Swords, Pentacles) 전체 수록.
- **백과사전급 데이터**: 카드당 책 한 페이지 분량의 방대한 서사, 심리학적 분석 및 정교한 상징 풀이 제공.
- **전문적 리딩 가이드**: 연애, 직업, 금전, 조언 분야별로 세분화된 정방향 및 **역방향(Reversed)** 리딩 데이터 구축.

### 2. 프리미엄 UI/UX
- **Dark Academia 디자인**: 신비롭고 고전적인 분위기의 사용자 인터페이스.
- **대형 독서 모달**: 방대한 텍스트를 책처럼 편안하게 읽을 수 있도록 최적화된 레이아웃.
- **반응형 그리드**: 모든 디바이스에서 최적의 카드 목록 뷰 제공.

### 3. 기술 스택
- **Frontend**: React, TypeScript, Vite, Vanilla CSS.
- **Backend**: Node.js, Express.
- **Data Architecture**: 대규모 텍스트 데이터를 효율적으로 관리하기 위한 슈트별 모듈화 구조.

## 🚀 시작하기

### 설치
```bash
npm install
```

### 실행
```bash
# API 서버 실행 (8787 포트)
npm run dev:api

# 웹 앱 실행 (5173 포트)
npm run dev:web
```

## 📂 프로젝트 구조
- `apps/api`: 타로 데이터 제공 및 리딩 엔진 (Node.js)
- `apps/web`: 세련된 타로 라이브러리 UI (React)
- `docs/`: 업데이트 내역 및 프로젝트 문서화
