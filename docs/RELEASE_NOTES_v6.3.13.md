# Release Notes v6.3.13 (2026-02-28)

## 목적
리딩 화면을 카드 중심 탐색에 맞게 재구성하여, 스프레드/기본설명/리포트를 한 화면에서 동시에 확인할 수 있도록 개선합니다.

## 핵심 변경
- 3분할 레이아웃 도입
  - 좌상단: 카드 스프레드
  - 좌하단: 기본 카드 설명
  - 우측: 운명의 리포트/아르카나 탐구(탭)
- 와이드 뷰 확장
  - 페이지 최대 폭: `1100px -> 1520px`
- 반응형 정비
  - 모바일/협소 화면에서 1열 스택으로 자동 전환

## 변경 파일
- `apps/web/src/pages/TarotMastery.tsx`
- `apps/web/src/pages/TarotMastery.module.css`
- `docs/CHANGELOG.md`
- `docs/RELEASE_NOTES_v6.3.13.md`

## 상세 구현
1. `TarotMastery.tsx`
- `workspaceGrid` 기준의 좌/우 패널 구조로 렌더링 트리 재배치.
- 좌하단 기본 카드 설명 패널 추가:
  - 포지션명, 카드명, 카드 요약(첫 문장) 노출.
- 질문 전 상태를 위한 좌상단 플레이스홀더 추가.

2. `TarotMastery.module.css`
- 2열 그리드(좌 46% / 우 54%) 및 좌측 내부 2행 그리드 구성.
- 우측 리포트 패널에 독립 경계/배경/오버플로 제어 적용.
- 좌하단 기본 설명 패널 스타일(`cardBasics*`) 추가.
- 모바일에서 단일 컬럼으로 전환되는 반응형 규칙 추가.

## 기대 효과
- 사용자가 카드 결과와 텍스트 해석을 오가며 확인하는 시간이 단축됨.
- 리딩 탭(보고서/탐구) 사용 시 화면 맥락 손실 감소.
- 데스크톱에서 정보량 대비 가독성 개선.
