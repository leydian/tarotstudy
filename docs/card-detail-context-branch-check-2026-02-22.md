# 카드 상세 맥락 분기 점검 리포트

작성일: 2026-02-22  
대상 API: `POST /api/cards/:cardId/explain`  
대상 카드: `major-0` (바보)  
레벨: `intermediate`

## 점검 목적
- 빈 질문 대비 맥락 질문(이직/재회/재정)에서 7개 섹션이 모두 달라지는지 확인
- 각 섹션이 최소 3줄 이상 유지되는지 확인

## 점검 컨텍스트
- 빈 질문: `""`
- 이직: `이직 타이밍이 언제가 좋을까?`
- 재회: `재회 연락을 해도 될까?`
- 재정: `재정 흐름과 지출 관리`

## 결과 요약
- 섹션 변화 수(빈 질문 대비):
  - 이직: 7/7
  - 재회: 7/7
  - 재정: 7/7
- 줄 수 기준:
  - `coreMeaning`, `symbolism`: 4줄
  - `upright`, `reversed`, `love`, `career`, `advice`: 3줄
- 결론: 분기 강도 기준(7개 섹션 전부 변화 + 섹션당 최소 3줄) 충족

## 섹션별 확인
- 변화 확인 섹션:
  - `coreMeaning`
  - `symbolism`
  - `upright`
  - `reversed`
  - `love`
  - `career`
  - `advice`

## 관찰 메모
- 교차 분기도 정상 동작합니다. 예를 들어, 이직 질문에서 `love` 섹션에 협업/경계 설정 문장이 들어가고, 재정 질문에서 `career` 섹션이 수입 안정화 맥락으로 보정됩니다.
