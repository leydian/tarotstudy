# 타로 코덱스 대규모 업데이트 보고서 (v2.2.2 - 학습 모드 콘텐츠 최적화 및 위치 정의 고도화)

## 1. 업데이트 개요 (v2.2.2)
학습 모드(Study Mode) 출력 시의 정보 중복을 제거하고, 리딩 위치별 정의를 더욱 구체화하여 사용자 경험의 완성도를 높였습니다.

## 2. 주요 개선 사항: 정보 구조 최적화 (Information Hierarchy)
- **학습 모드 메시지 정제**:
    - 이미 리딩 과정에서 노출된 카드 해석 텍스트를 학습 모드 메시지에서 제외하여 중복을 제거했습니다.
    - 대신 카드의 '본질적 상징 분석'과 '핵심 키워드'를 전면에 배치하여 학습 효과를 극대화했습니다.
- **위치 정의(Position Definitions) 확장**:
    - [과거, 현재, 미래, 기반, 핵심] 등 다양한 스프레드 위치에 대한 고유한 정의를 추가하여, 기본 문구("운명의 중요한 단계입니다")의 반복 노출을 최소화했습니다.
- **가독성 및 레이블 개선**:
    - 불필요한 기호와 레이블을 정리하고, 핵심 내용을 더 직관적으로 전달할 수 있도록 텍스트 형식을 조정했습니다.

## 1. 업데이트 개요 (v2.2.1)
리딩 결과 확인 중에도 뽑힌 카드의 시각적 상징을 지속적으로 참고할 수 있도록, 카드 드로우 영역의 유지 로직을 개선했습니다.

## 2. 주요 개선 사항: 시각적 연속성 (Visual Continuity)
- **카드 이미지 상시 노출**:
    - 리딩 결과(Master Report)가 출력된 이후에도 카드들이 사라지지 않고 질문 바로 아래에 고정되도록 수정했습니다.
    - 사용자는 사서의 텍스트 리딩을 읽으면서 동시에 카드의 이미지를 실시간으로 대조하며 통찰을 얻을 수 있습니다.
- **드로우 영역 디자인 최적화**:
    - 리딩 결과와 조화롭게 어우러지도록 카드 드로우 섹션의 배경색(`rgba(212, 175, 55, 0.03)`)과 크기를 조정하여 시각적 피로도를 줄였습니다.
