# 타로 코덱스 대규모 업데이트 보고서 (v1.3.0 - 전문 스프레드 대확장)

## 1. 업데이트 개요
사용자의 고민 유형(연애, 커리어, 미래 흐름 등)을 정교하게 분류하고, 각 상황에 최적화된 **전문 타로 스프레드 3종을 신규 도입**했습니다. 이로써 1장부터 12장까지 총 9가지의 다양한 스프레드 체계를 갖추게 되었습니다.

## 2. 신규 전문 스프레드 도입
- **관계의 거울 (Relationship - 7장)**: 나와 상대방의 현재 상태, 숨겨진 속마음, 관계의 장애물 및 최종 유대감을 입체적으로 분석.
- **커리어 패스 (Career Path - 5장)**: 현재 실력 진단, 잠재적 재능 발견, 현실적 장애물 극복 및 성공의 열쇠 제시.
- **행운의 말편자 (Horseshoe - 7장)**: 과거-현재-미래를 잇는 거시적 흐름 분석 및 타인의 시선, 조언을 포함한 종합 전망 제공.

## 3. 챗봇 지능형 의도 분석 (Intent Engine v2)
- **키워드 정밀 매칭**: 
  *   연애/재회/속마음 관련 질문 시 -> **관계의 거울** 자동 선택.
  *   이직/취업/승진 관련 질문 시 -> **커리어 패스** 자동 선택.
  *   전망/흐름/운세 관련 질문 시 -> **행운의 말편자** 자동 선택.
- **동적 카드 장수 조절**: 질문의 무게와 주제에 따라 카드 장수(1, 2, 3, 5, 7, 10, 12장)를 실시간으로 결정하여 리딩의 깊이 조절.

## 4. 엔진 및 UX 안정화
- **대량 리딩 최적화**: 7장 이상의 리딩에서도 서사가 끊기지 않도록 요약 및 총평 로직 보강.
- **UI 일관성 유지**: 모든 스프레드에서 카드 크기와 레이블 디자인을 통일하여 시각적 안정감 확보.
- **에러 핸들링**: 비동기 메시지 전달 과정에서의 구문 오류 및 끊김 현상 완벽 해결.
