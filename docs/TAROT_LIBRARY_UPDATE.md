# 타로 코덱스 대규모 업데이트 보고서 (v2.4.1 - 카드 도서관 필터 정합성 강화)

## 1. 업데이트 개요 (v2.4.1)
카드 도서관의 마이너 아르카나 필터링 오류를 수정하여 데이터 정합성을 확보하고, 사용자 경험을 개선했습니다.

## 2. 주요 개선 사항: 데이터 정합성 및 필터링 최적화

### 2.1. 마이너 아르카나 필터링 로직 수정 (Bug Fix)
- **ID 접두사 매칭 도입**: 기존에 전체 수트 이름(wand, cup 등)으로 검색하던 로직이 실제 데이터 ID(w01, c01 등)와 일치하지 않아 발생하던 문제를 해결했습니다.
- **하위 호환성 유지**: `startsWith`를 통한 접두사 매칭과 기존 `includes` 방식을 병행하여 데이터 구조 변화에도 유연하게 대응하도록 설계했습니다.
- **필터 대상**: 완드(Wands), 컵(Cups), 소드(Swords), 펜타클(Pentacles) 등 4종의 마이너 아르카나 전체에 적용되었습니다.

### 2.2. 아키텍처 현대화 (v2.4.0)
- **컴포넌트 모듈화**: 비대해진 페이지 컴포넌트에서 `TarotCard`, `MessageBubble` 등을 분리하여 독립적인 컴포넌트로 재구축했습니다.
- **서비스 레이어 도입**: API 통신 로직을 `tarotService.ts`로 캡슐화하여 비즈니스 로직과 데이터 요청을 분리했습니다.
- **엄격한 타입 시스템**: 프로젝트 전반에 걸쳐 사용되는 데이터 구조를 `tarot.ts` 타입 파일로 정의하여 런타임 에러를 방지하고 유지보수 효율을 높였습니다.
