# 타로 코덱스 대규모 업데이트 보고서 (v1.7.0 - Arcane Master 정밀 분석 엔진 도입)

## 1. 업데이트 개요
리딩의 논리적 일관성을 확보하고 표현의 다양성을 극대화한 **'정밀 분석형'** 시스템을 구축했습니다. 리딩 엔진을 V4.0(Arcane Master)으로 업그레이드하여, 예스노 판정의 정확도를 높이고 가변적인 서사를 제공합니다.

## 2. 상담 엔진 정밀화 (Arcane Master v4.0)
- **예스노 판정 알고리즘 고도화**: 전차(The Chariot), 광대(The Fool) 등 실질적 긍정 의미가 강한 카드들을 점수 체계에 대폭 추가하여 리딩 내용과 판정 결과(YES/NO/MAYBE) 사이의 논리적 무결성을 확보했습니다.
- **가변 비유 시스템 (Metaphor Bank)**: 조언 섹션에 고정되어 있던 비유(등불, 봄볕 등)를 5가지 이상의 세트로 확장하여 무작위로 적용합니다. 이를 통해 반복적인 상담에서도 매번 새로운 느낌을 줍니다.
- **현대적 문체 전환 로직**: 질문 내의 단어(게임, 코인, 이직 등) 및 영문 혼용 여부를 감지하여, 지나치게 고전적인 은유 대신 직설적이고 분석적인 말투를 사용하는 '현대적 모드'를 도입했습니다.
- **스프레드 레이아웃 확장**: 켈틱 크로스(10장)와 연간 운세(12장) 스프레드의 각 위치별 의미를 완벽하게 정의하여 해설의 깊이를 완성했습니다.

## 3. UI/UX 품질 개선
- **카드 이름 중복 노출 해결**: 카드 이미지의 `alt` 텍스트와 하단 라벨이 복사 또는 드래그 시 중복으로 잡히던 현상을 수정하고, 시각적으로 깔끔한 캡션 스타일을 적용했습니다.
- **텍스트 가독성 최적화**: 분석 보고서 형식의 서두와 감성적인 결론을 조화롭게 배치하여 정보 전달력과 정서적 만족도를 동시에 높였습니다.

## 4. 품질 관리 및 배포
- **리딩 모순 해결**: 긍정 카드 중심의 리딩에서 부정적 판정이 나오던 로직 버그를 전면 수정했습니다.
- **형상 관리**: 최신 엔진 로직을 `main` 브랜치에 통합하고 Vercel 배포 파이프라인을 점검했습니다.
