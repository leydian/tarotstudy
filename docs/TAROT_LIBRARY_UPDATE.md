# 타로 코덱스 대규모 업데이트 보고서 (v1.4.0 - 인프라 및 상담 고도화)

## 1. 업데이트 개요
사용자 맞춤형 전문 상담 기능을 대폭 확장하고, 실제 도메인(`창영소타로.xyz`) 배포를 위한 Vercel 클라우드 인프라 설정을 최적화했습니다. 이제 모바일과 데스크톱 어디서나 안정적인 타로 상담 서비스를 제공합니다.

## 2. 배포 및 인프라 최적화 (Vercel)
- **모노레포 라우팅**: `/api` 경로는 Node.js 백엔드로, 그 외 모든 경로는 React 프론트엔드로 연결되는 지능형 라우팅 구조 구축.
- **빌드 파이프라인**: Vercel의 빌드 규격에 맞춘 `vercel-build` 스크립트 도입 및 정적 결과물(`dist`) 경로 최적화.
- **도메인 호환성**: 모든 API 호출을 상대 경로로 전환하여 커슬텀 도메인 및 한글 도메인(Punycode) 환경에서 무결성 확보.

## 3. 전문 스프레드 라이브러리 완성
- **관계의 거울 (7장)**: 두 사람 사이의 에너지, 속마음, 장애물을 분석하는 심층 애정운 기능.
- **커리어 패스 (5장)**: 직무 역량, 잠재력, 현실적 난관 및 성공 전략을 제시하는 직업운 특화 기능.
- **행운의 말편자 (7장)**: 과거-현재-미래 및 주변 환경을 포괄하는 거시적 전망 기능.
- **기존 스프레드 보강**: 켈틱 크로스(10장), 연간 운세(12장), 양자택일(2장) 등 총 9종의 스프레드 라인업 구축.

## 4. 상담 엔진 지능화 (Adaptive Engine v3.8)
- **양자택일 정밀 감지**: 한국어 종결 어미(`~까?`) 반복 및 선택 접속사 분석을 통한 자동 비교 모드 진입.
- **상황별 가변 리딩**: 질문의 주제(연애, 이직, 금전)를 스스로 판별하여 최적의 분야별 해설 자동 조합.
- **에러 복구 시스템**: 비동기 대화 흐름 중 발생할 수 있는 데이터 누락 방지 및 이미지 로딩 실패 시 자동 대응 로직 강화.

## 5. 프로젝트 정비
- 브랜치 통합: `publish/no-workflow-v2`의 최신 성과를 `main`으로 완전 통합.
- 보안 강화: 워크플로우 권한 이슈 해결 및 불필요한 설정 파일 정리.
