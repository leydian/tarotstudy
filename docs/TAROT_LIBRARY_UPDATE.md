# 타로 코덱스 대규모 업데이트 보고서 (v2.1.4 - 아르카나 성소 레이아웃 확장 및 지능형 학습 모드 강화)

## 1. 업데이트 개요 (v2.1.4)
'아르카나 성소'의 상담 환경을 더욱 쾌적하게 개선하고, 리딩 결과와 연계된 학습 경험을 극대화하기 위해 레이아웃 확장 및 지능형 학습 모드(Study Mode)를 고도화했습니다.

## 2. 주요 개선 사항: 리딩 환경 최적화 (Layout Expansion)
- **와이드 레이아웃 확장**: 
    - 성소 페이지의 전체 너비를 **1600px에서 1800px로 확장**하여 더욱 넓은 시야를 제공합니다.
    - 왼쪽 채팅창 너비를 소폭 축소(500px -> 450px)하여, 오른쪽의 리딩 결과 및 스프레드 영역을 최대로 확보했습니다.
- **아르카나의 지침 레이아웃 변경**: 
    - 기존 좌우 2열 그리드 방식을 **상하 1열 리스트 방식**으로 변경하여, 모바일과 데스크톱 모두에서 조언의 흐름을 방해 없이 읽을 수 있도록 개선했습니다.

## 3. 지능형 학습 모드 고도화 (Intelligent Study Mode)
- **실시간 학습 연동 (Real-time Insights)**: 
    - 카드를 뒤집을 때 학습 모드가 활성화되어 있다면, 일반 해석뿐만 아니라 해당 위치의 상징적 의미, 카드의 심층 상징, 키워드를 즉시 채팅창에 출력합니다.
- **일괄 학습 갱신 (Batch Insight Generation)**: 
    - 리딩 결과 화면에서 '학습 모드 ON' 버튼을 누르면, 이번 리딩에서 뽑힌 모든 카드에 대해 순차적으로 심층 분석 리포트를 채팅창에 자동 갱신합니다.
- **정보 구성 정교화**: 
    - 각 분석 메시지는 [심화 학습: 카드명] 형식으로 제공되며, 위치의 역할, 상징 분석, 키워드, 이번 리딩에서의 구체적 해석을 모두 포함합니다.

## 4. UI/UX 정밀 조정
- **간격 및 패딩 최적화**: 레이아웃 확장에 맞춰 요소 간 간격(Gap)과 패딩을 재조정하여 시각적 밸런스를 맞췄습니다.
- **상태 관리 개선**: 리딩 단계에 따른 학습 모드 버튼의 노출과 작동 방식을 더욱 직관적으로 개선했습니다.
