# 타로 코덱스 대규모 업데이트 보고서 (v2.3.3 - 이미지 로딩 안정화 및 라이선스 명시)

## 1. 업데이트 개요 (v2.3.3)
일부 타로 카드 이미지가 정상적으로 불러와지지 않는 문제를 해결하기 위해 이미지 로딩 방식을 전면 개편하고, 저작권 보호 및 투명성 확보를 위해 이미지 라이선스 정보를 명시했습니다.

## 2. 주요 개선 사항: 이미지 인프라 및 신뢰성 (Image Infrastructure)

### 2.1. 이미지 로딩 방식 안정화 (Robust Image Loading)
- **Wikimedia Special:FilePath 도입**: 기존의 복잡한 해시 기반 URL(`upload.wikimedia.org/.../a/ab/...`) 대신, 위키미디어 공용의 공식 파일 경로 시스템(`Special:FilePath`)을 사용하여 모든 카드 이미지를 안전하게 불러오도록 수정했습니다.
- **자동 리다이렉션**: 파일명만 정확하면 시스템이 자동으로 최적의 서버 경로를 찾아주므로, 향후 위키미디어의 서버 구조 변경에도 유연하게 대응할 수 있습니다.
- **78장 전수 조사**: 메이저 아르카나부터 마이너 아르카나(지팡이, 컵, 검, 펜타클)까지 모든 카드의 이미지 경로를 최신화했습니다.

### 2.2. 라이선스 및 출처 명시 (License Disclosure)
- **퍼블릭 도메인 명시**: 본 서비스에서 사용된 **Rider-Waite-Smith Tarot (1909)** 이미지가 저작권이 만료된 퍼블릭 도메인임을 명확히 안내했습니다.
- **전역 푸터(Footer) 반영**: 모든 페이지 하단에 이미지 출처와 라이선스 정보를 추가하여 서비스의 신뢰도를 높였습니다.
- **카드 도서관 상세 안내**: '카드 도서관' 페이지 하단에 타로 덱 제작자(아서 에드워드 웨이트, 파멜라 콜먼 스미스)와 이미지 출처(Wikimedia Commons)를 상세히 기록했습니다.
