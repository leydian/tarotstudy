# 타로 코덱스 대규모 업데이트 보고서 (v2.1.0 - 아르카나 성소 통합 시스템 도입)

## 1. 업데이트 개요 (v2.1.0)
기능별로 분산되어 있던 **운명 읽기(Reading)**, **챗봇 리딩(Chat)**, **운명 학습(Study)** 페이지를 하나의 단일 인터페이스로 통합한 **'아르카나 성소(Arcane Sanctum)'** 시스템을 구축했습니다. 사용자는 탭 이동 없이 하나의 여정 속에서 상담, 시각적 리딩, 심화 학습을 동시에 경험할 수 있습니다.

## 2. 통합 탭: 아르카나 성소 (TarotMastery.tsx)
- **듀얼 레이아웃(Dual Layout) 아키텍처**:
    - **왼쪽 사이드바(Chat Panel)**: 챗봇 사서와의 실시간 대화 기록을 관리하며, 상담의 맥락을 끊임없이 이어줍니다.
    - **오른쪽 작업 영역(Ritual Panel)**: 웅장한 시각적 공간에서 카드 스프레드를 펼치고, 사용자가 직접 카드를 뒤집는(Flip) 리추얼을 진행합니다.
- **실시간 리딩 인터랙션 (Real-time Interpretation)**: 사용자가 오른쪽 패널에서 카드를 하나씩 뒤집을 때마다, 왼쪽 챗봇이 해당 카드의 **위치별 역할**과 **구체적 해석**을 즉각적으로 설명해 줍니다.
- **온디맨드 학습 모드 (On-demand Study Mode)**: 리딩 완료 후 상단의 **'학습 모드'** 스위치를 켜면, 펼쳐진 카드에 마우스를 올려 해당 자리에 대한 **철학적 정의**와 **상징 분석** 팝오버를 즉시 확인할 수 있습니다.

## 3. UI/UX 간소화 및 효율화
- **내비게이션 단일화**: [운명 읽기, 챗봇 리딩, 운명 학습] 3개 메뉴를 **[아르카나 성소]**라는 하나의 강력한 통합 탭으로 통합하여 사용자 인터페이스를 단순화했습니다.
- **상태 관리 통합 (State Unification)**: 대화 로그, 배열 데이터, 해석 결과, 학습 상태를 하나의 컴포넌트 내부에서 효율적으로 관리하여 페이지 전환 시 발생할 수 있는 데이터 손실을 원천 차단했습니다.
- **지능형 질문 인지 강화**: V5.1 엔진의 질문 분석 로직을 통해 사용자가 어떤 질문을 던져도 최적의 스프레드(1~12장)를 자동으로 매칭하고 해설합니다.
