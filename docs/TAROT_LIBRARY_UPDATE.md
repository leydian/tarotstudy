# 타로 코덱스 대규모 업데이트 보고서 (v2.2.3 - 한국어 문법 교정 및 복합 레이블 정의 보완)

## 1. 업데이트 개요 (v2.2.3)
리딩 결과의 문법적 완성도를 높이기 위해 한국어 조사(와/과) 선택 로직을 개선하고, 복합적인 스프레드 위치 레이블에 대한 정의를 보완했습니다.

## 2. 주요 개선 사항: 언어적 정교화 (Linguistic Refinement)
- **지능형 조사 선택 로직 (Smart Josa Selection)**:
    - 비유적 조언 섹션에서 앞 단어의 받침 유무를 판별하여 '와/과'를 적절하게 선택하도록 API 엔진을 고도화했습니다.
    - 예: "등불와 같습니다" -> "등불과 같습니다", "봄볕처럼" -> "봄볕 같이"
- **복합 레이블 정의 추가**:
    - [과거/기반, 현재/중심, 미래/흐름] 등 다수의 정보를 포함하는 위치 레이블에 대한 고유 설명을 추가했습니다.
    - 학습 모드에서 더욱 풍부한 위치 정보를 제공하여 사용자에게 명확한 컨텍스트를 전달합니다.

# 타로 코덱스 대규모 업데이트 보고서 (v2.2.2 - 학습 모드 콘텐츠 최적화 및 위치 정의 고도화)

---

## 1. 업데이트 개요 (v2.2.1)
리딩 결과 확인 중에도 뽑힌 카드의 시각적 상징을 지속적으로 참고할 수 있도록, 카드 드로우 영역의 유지 로직을 개선했습니다.

## 2. 주요 개선 사항: 시각적 연속성 (Visual Continuity)
- **카드 이미지 상시 노출**:
    - 리딩 결과(Master Report)가 출력된 이후에도 카드들이 사라지지 않고 질문 바로 아래에 고정되도록 수정했습니다.
    - 사용자는 사서의 텍스트 리딩을 읽으면서 동시에 카드의 이미지를 실시간으로 대조하며 통찰을 얻을 수 있습니다.
- **드로우 영역 디자인 최적화**:
    - 리딩 결과와 조화롭게 어우러지도록 카드 드로우 섹션의 배경색(`rgba(212, 175, 55, 0.03)`)과 크기를 조정하여 시각적 피로도를 줄였습니다.
