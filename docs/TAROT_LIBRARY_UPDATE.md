# 타로 코덱스 대규모 업데이트 보고서 (v2.2.5 - 일상 질문 대응 톤앤매너 최적화 및 서사 중복 제거)

## 1. 업데이트 개요 (v2.2.5)
가벼운 일상적인 질문(예: 커피, 메뉴 선택 등)에 대해 AI 사서가 지나치게 무겁거나 철학적인 답변을 내놓는 '톤앤매너 불일치' 문제를 해결하고, 리포트 내의 텍스트 중복을 제거하여 가독성을 높였습니다.

## 2. 주요 개선 사항: 상황 인지형 리딩 (Context-Aware Reading)
- **가벼운 질문 전용 필터링 (Light Mode Tone)**:
    - `isLight` 감지 로직이 활성화될 경우, 카드의 심오하고 철학적인 설명 대신 핵심 요약 위주의 간결하고 부드러운 문체를 사용하도록 개선했습니다.
    - "영적인 갈증"이나 "비극적인 정서"와 같은 무거운 표현들을 일상적인 고민에 맞는 적절한 비유로 완화했습니다.
- **서사 분석 중복 제거 (Redundancy Reduction)**:
    - [운명의 서사 분석] 섹션에서 개별 카드 해석을 그대로 반복하던 문제를 해결했습니다.
    - 각 카드의 핵심 의미만 추출하여 하나의 유기적인 흐름으로 연결하는 '요약형 서사' 방식을 도입하여 리포트의 길이를 최적화했습니다.
- **비유적 언어의 현대화**:
    - 지나치게 문어체이거나 고풍스러운 표현들을 현대적인 관점에서 이해하기 쉬운 문장으로 다듬었습니다.
