# 타로 코덱스 대규모 업데이트 보고서 (v1.5.0 - Arcane Insight 엔진 고도화)

## 1. 업데이트 개요
단순 키워드 매칭을 넘어선 **'심층 역학 분석'** 시스템을 도입했습니다. 리딩 엔진을 V3.8(Arcane Insight)로 업그레이드하여, 카드 간의 상관관계와 질문자의 에너지 밸런스를 입체적으로 분석합니다.

## 2. 상담 엔진 고도화 (Arcane Insight v3.8)
- **원소 밸런스 분석 (Elemental Analysis)**: 78장 카드의 4대 슈트(불, 물, 공기, 흙) 및 메이저(정신) 비중을 계산하여 질문자의 전반적인 에너지 상태를 진단합니다.
- **스프레드 위치학 적용 (Positional Logic)**: 카드 장수에 따라 '과거-현재-미래', '나-상대방-관계' 등 위치별 고유 의미를 부여하여 해설의 정밀도를 높였습니다.
- **운명적 조합 감지 (Card Combinations)**: 특정 카드 쌍(예: 죽음+태양)이 나타날 경우 발생하는 '시너지 서사'를 감지하여 입체적인 리딩을 제공합니다.
- **상황 인지형 페르소나 (Persona Awareness)**: 질문 내용을 분석하여 학생, 직장인, 커플, 솔로 등 질문자의 상황에 최적화된 문체와 조언을 제공합니다.

## 3. UI/UX 개선 (Chat Reading)
- **중복 렌더링 해결**: 카드 출력 시 고유 키(Key) 처리 및 데이터 정돈을 통해 이름이 중복 출력되던 문제를 해결했습니다.
- **상황 적응형 톤앤매너**: 질문의 무게감(가벼운 질문 vs 무거운 질문)에 따라 마스터의 어조가 자동으로 변화합니다.
- **카테고리 자동 전환**: 질문 의도에 따라 연애, 직업, 금전 카테고리를 자동 선택하여 전문 해설을 제공합니다.

## 4. 인프라 및 배포 (Vercel)
- **Punycode 도메인 안정화**: `타로박사.xyz` (xn--2o2b11o1tdn2hc9f.xyz) 도메인 연결 설정 가이드 및 배포 안정성 확보.
- **루트 빌드 스크립트 정비**: 모노레포 구조에서 Vercel 자동 빌드가 원활하도록 루트 `package.json`의 빌드 파이프라인 최적화.
