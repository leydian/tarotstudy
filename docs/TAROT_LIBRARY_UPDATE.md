# 타로 코덱스 대규모 업데이트 보고서 (v1.9.1 - 맥락 기반 연계 학습 고도화)

## 1. 업데이트 개요 (v1.9.1)
'운명 학습(Study Reading)' 시스템에 **'리딩 맥락 보존(Context Preservation)'** 기술을 적용했습니다. 사용자는 단순한 카드의 사전적 의미를 넘어, "이번 질문에서 이 카드가 왜 과거/현재/미래의 자리에 배치되었는지"에 대한 실질적인 상담 데이터와 상징 체계를 통합적으로 학습할 수 있습니다.

## 2. 주요 개선 사항: 맥락 기반 학습 (Contextual Learning)
- **리딩 데이터 실시간 매핑**: 상담 시 생성된 각 카드의 **위치별 역할(예: [미래의 빛], [장애물] 등)**과 **구체적인 해석 문구(Evidence)**를 학습 세션에 즉시 연동합니다.
- **입체적 학습 구조**: 학습 모드 실행 시 다음과 같은 3단계 레이아웃으로 정보를 제공합니다.
    1.  **[실전 분석]**: 이번 질문에서 해당 카드가 수행한 역할과 실제 해석 내용.
    2.  **[상징 분석]**: 카드 고유의 상징물(Symbolism)에 대한 깊이 있는 분석.
    3.  **[서사 및 키워드]**: 카드의 전체적인 스토리와 핵심 키워드 정리.
- **세션 상태 고도화**: `sessionReading` 상태 관리를 통해 API 재호출 없이도 방금 진행한 상담의 모든 세부 데이터를 학습창에 즉각적으로 풀어내는 비동기 로직을 구현했습니다.

## 3. 학습자 중심 인터페이스 강화
- **명확한 구분선**: 실전 해석과 사전적 지식을 구분하여 사용자가 '해석의 원리'를 스스로 터득할 수 있도록 시각적 구분을 강화했습니다.
- **학습 유도 메시지 정교화**: "각 카드가 이번 질문에서 어떤 역할을 했는지 공부해 보시겠습니까?"와 같은 구체적인 제안을 통해 학습 동기를 부여합니다.
