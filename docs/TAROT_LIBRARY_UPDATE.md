# 타로 카드 도서관 대규모 업데이트 보고서 (2026-02-25)

## 1. 업데이트 개요
기존의 단순 키워드 중심 카드 데이터를 전문가 수준의 심층 리딩 데이터로 전면 교체하였습니다. 78장 전체 카드에 대해 '책 한 페이지' 분량의 상세 해설과 정/역방향 리딩 가이드를 구축했습니다.

## 2. 데이터 구조 개선
데이터의 방대함을 고려하여 파일 분리 구조를 채택, 데이터 누락을 방지하고 관리 효율을 높였습니다.

- **데이터 파일 위치**: `apps/api/src/data/`
  - `major.js`: 메이저 아르카나 (22장)
  - `wands.js`: 지팡이 아르카나 (14장)
  - `cups.js`: 컵 아르카나 (14장)
  - `swords.js`: 검 아르카나 (14장)
  - `pents.js`: 펜타클 아르카나 (14장)
  - `cards.js`: 위 5개 파일을 통합하여 최종 export

- **카드별 데이터 필드**:
  - `id`: 고유 식별자 (m00, w01 등)
  - `name / nameKo`: 영문명 및 국문명
  - `keywords`: 리딩 핵심 키워드 리스트
  - `summary`: 한 줄 요약
  - `image`: RWS(Universal Waite) 버전 이미지 URL
  - `description`: 카드의 상징적 배경 및 심리학적 서사 (장문)
  - `symbolism`: 이미지 속 개별 상징 분석
  - `meanings`: 정방향 리딩 (연애, 직업, 금전, 조언)
  - `reversed`: 역방향 리딩 (요약 및 각 분야별 경고/가이드)

## 3. UI/UX 고도화 사항
`apps/web/src/pages/Cards.tsx`를 수정하여 고밀도 데이터를 효과적으로 보여주도록 개선했습니다.

- **대형 독서 모달**: 90vh 높이의 대형 모달을 통해 긴 텍스트의 가독성 확보
- **정/역방향 대비 레이아웃**: 초록색(정방향)과 빨간색(역방향) 섹션을 구분하여 한눈에 비교 가능
- **시각적 피드백**: 카드 호버 시 부양 효과 및 골드 테두리 강조로 신비로운 분위기 조성
- **Lazy Loading**: 78장의 이미지를 효율적으로 로드하도록 최적화
