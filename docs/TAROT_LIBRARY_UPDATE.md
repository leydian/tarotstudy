# 타로 코덱스 대규모 업데이트 보고서 (v1.9.0 - 학습 중심 리딩 시스템 도입)

## 1. 업데이트 개요 (v1.9.0)
사용자가 운세 결과를 확인하는 것에서 나아가, 뽑힌 카드의 상징과 철학을 깊이 있게 공부할 수 있는 **'학습 연계형 리딩(Study Reading)'** 시스템을 새롭게 도입했습니다. 이를 통해 단순 점술 서비스를 넘어 타로 지식을 체계적으로 습득할 수 있는 교육적 가치를 더했습니다.

## 2. 신규 기능: 운명 학습 (Study Reading)
- **대화형 학습 모드 (Interactive Learning)**: 리딩이 완료된 시점에 `[네, 깊이 배우고 싶어요]` 버튼을 통해 즉시 학습 모드로 전환할 수 있습니다. 봇이 뽑힌 카드들의 **상징 분석(Symbolism)**과 **상세 서사(Description)**를 대화창에 차례로 풀어주며 깊이 있는 이해를 돕습니다.
- **카드 도서관 실시간 연동**: 학습 중인 카드를 클릭하면 '카드 도서관'의 해당 카드 상세 페이지로 즉시 이동하여 더 넓은 맥락의 정보를 확인할 수 있습니다.
- **학습용 특화 UI**: 기존 챗봇보다 더 넓은 말풍선과 가독성 높은 줄바꿈, 그리고 강조된 황금색 메뉴를 통해 '탐구와 지혜'의 분위기를 조성했습니다.

## 3. 라우팅 및 내비게이션 통합
- **독립 경로 확보**: `/study-reading` 경로를 신설하여 기존의 단순 상담용 `/chat-reading`과 독립적으로 운영됩니다.
- **GNB(Global Navigation Bar) 갱신**: 헤더 메뉴에 **[운명 학습]** 링크를 추가하고 강조 색상을 적용하여 사용자의 접근성을 높였습니다.

## 4. 기술적 개선
- **세션 기반 카드 관리**: 리딩 중 뽑힌 카드들을 세션 상태로 관리하여, 학습 모드 전환 시 별도의 API 호출 없이 즉각적으로 상세 데이터를 매핑하도록 설계했습니다.
- **비동기 단계적 출력**: 학습 내용이 너무 길어 가독성을 해치지 않도록, 각 카드별 상세 설명을 비동기 타이머를 통해 순차적으로 출력하는 연출을 적용했습니다.
