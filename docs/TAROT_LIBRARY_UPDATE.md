# 타로 코덱스 대규모 업데이트 보고서 (v1.8.1 - Omniscient Master 미세 조정 및 안정화)

## 1. 업데이트 개요 (V5.1)
리딩 엔진을 **V5.1 (Omniscient Master - 미세 조정 버전)**으로 업데이트했습니다. V5.0에서 도입된 전지적 인지 기능을 바탕으로, 실제 사용자들의 다양한 질문 패턴(특히 구어체 양자택일)을 더 정교하게 분석하고, 모든 상담에 대해 명확한 결론을 제공하도록 개선했습니다.

## 2. 엔진 미세 조정 사항 (V5.1)
- **정규식 기반 양자택일 감지 (Regex-based Choice Detection)**: "A 마실까 B 마실까?", "갈까 말까?" 등 구어체에서 반복되는 의문형 패턴을 인식하는 `binaryPattern` 정규식을 도입했습니다. 이를 통해 단순 키워드 매칭보다 훨씬 높은 정확도로 선택형 질문을 판별합니다.
- **결론 출력 강제화 (Mandatory Verdicts)**: 질문의 유형(연애, 금전, 가벼운 질문 등)과 관계없이 모든 리딩 결과의 마지막에 `[운명의 나침반]` 판정이 반드시 포함되도록 로직을 수정했습니다. 상담의 종결감을 높이고 사용자에게 명확한 가이드를 제공합니다.
- **문장 조립 로직 최적화**: 인사말, 원소 테마, 브릿지 문장 간의 결합 시 발생하던 어색한 줄바꿈과 중복 마침표 문제를 해결하여 리딩 텍스트의 가독성을 극대화했습니다.

## 3. 기존 V5.0 주요 기능 (복습)
- **감정 공감 지능**: 슬픔/불안 키워드 감지 시 사서의 인사말 자동 전환 및 특별 위로 조언 제공.
- **다자택일 랭킹 시스템**: 3개 이상의 선택지 중 카드 점수에 따른 1순위 추천 기능.
- **시기 예측 모듈**: 4원소 기반의 구체적 시기(수일 내, 봄/여름/가을/겨울 등) 판독.
- **스마트 컨텍스트 변환기**: 학업, 건강 등 특수 카테고리에 대한 실시간 문맥 치환 해설.
