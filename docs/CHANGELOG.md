# Changelog

## [2026-02-28]

### AI 리딩 엔진 고도화 (Reading V3 - Narrator V5.1)

#### 변경 사항
- **서사적 위치(Position) 설명 개선**:
    - "운명의 X단계"와 같은 기계적인 순서 표현을 실제 스프레드 위치 라벨(예: 잠재적 재능, 숨겨진 기회)로 대체.
    - `posIntro` 생성 시 다양한 연결 어구(`이어서`, `시선을 옮겨`, `비추는 곳에는` 등)를 사용하여 반복성 감소.
- **문법적 자연스러움 강화**:
    - **조사 처리 헬퍼(`getJosa`) 확장**: '을/를', '은/는' 처리를 추가하여 문맥에 맞는 정확한 한국어 조사 선택.
    - **조언 문구 연결 최적화**: 명령형 조언 뒤에 "라는 사서의 조언을 가슴 깊이 새긴다면" 등의 연결구를 도입하여 문법적 단절 해결.
    - **하단 Action 섹션 개선**: 조언과 비유 사이의 연결을 "하여,", "한다면," 등으로 부드럽게 리팩토링.
- **도입부 및 서사 흐름 정리**:
    - `elementTheme`와 `bridge` 사이의 중복된 운명 관련 표현(예: "운명의 거대한...")을 제거하고 서사의 단계적 전개 강조.
    - 리딩 전체의 몰입감을 위해 단조로운 종결 어미와 문장 구조를 다양화.

### UI/UX 개선 및 레이아웃 리팩토링 (TarotMastery)

#### 변경 사항
- **레이아웃 구조 개편**:
    - `mainContent`, `topSpreadArea`, `messagesContainer` 도입으로 페이지 구조 체계화.
    - 상단 스프레드 영역(`topSpreadArea`)을 고정형(Sticky-like)으로 변경하여 사용자 경험 개선.
- **탭 시스템 도입**:
    - 기존 토글 방식에서 탭 시스템(`tabContainer`, `tabHeader`)으로 전환.
    - Study 모드와 Reading 결과를 명확하게 분리하여 탐색 효율성 증대.
- **학습 모드(Study Mode) UI 최적화**:
    - `studyGrid` 및 `studyCard` 기반의 현대적인 카드 레이아웃 적용.
    - 카드 정보 가독성 향상 및 애니메이션 효과 추가.
- **스타일 정리**:
    - 불필요한 레거시 스타일(`studyToggle`, 구형 `spreadCanvas`) 제거.
    - 메시지 영역 스크롤바 및 간격 세밀 조정.
