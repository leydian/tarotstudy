# 연간운세 회귀 체크리스트

작성일: 2026-02-22

## 목적
- 연간운세 출력의 구조/톤 회귀를 빠르게 감지한다.
- 문서 점검 항목(분기/월 정합성, 4분기 마무리 톤)을 실행형 체크로 고정한다.

## 자동 점검
- 명령: `npm run qa:yearly-fortune`
- 입력 케이스: `scripts/yearly-fortune-regression-cases.json`
- 케이스 구성: 총 18건 (커리어 6건 + 관계 5건 + 재정 4건 + 일반 3건)
- 검증 항목:
  - `총평 → 분기별 운세 → 월별 운세` 섹션 존재
  - 분기 라벨 4개(`1분기`~`4분기`) 존재
  - 월별 내러티브 12개(`1월`~`12월`) 존재
  - `연말` 표현 존재(4분기 마무리 톤 가드)
  - 커리어 질문에서 취직 시기 마무리 문구 포함 여부
  - `언제`형 시기 질문에서 시간 표현 존재(월/분기/연말 등)
  - 커리어 질문에서 도메인 단어 2회 이상 포함(지원/면접/이력서/포트폴리오/오퍼/협상 등)
  - 커리어 시기 질문 최종 문단이 `"언제 + 무엇"` 형태를 포함하는지 확인

## 수동 UX 점검
- 모바일/데스크톱에서 스프레드 슬롯 카드 클릭 시 `/cards/:cardId` 이동 확인
- 연간운세 질문 3종(취업/연애/재정)에서 분기/월 문장 자연스러움 샘플 점검
- 4분기 텍스트가 새 확장보다 정리/전환 중심인지 확인
